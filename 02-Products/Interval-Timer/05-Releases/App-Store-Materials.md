## 📸 截图要求

**必须尺寸**（选择 2 个即可，推荐 6.5" 和 5.5"）：

**数量**：3-10 张

**内容建议**：
1. **首页**：展示快速开始区域和预设方案

3. **计时器页**：展示休息中状态
4. **方案页**：展示自定义和预设方案列表
5. **统计页**：展示训练数据和趋势图

**设计要点**：
- 使用真实数据（不要用占位符）
- 展示核心功能
- 界面干净、美观
- 添加设备边框（可选，使用 [App Screenshot Builder](https://appscreenshots.io/)）

```
Interval Timer - 简易间歇训练计时器

让你的健身训练更科学、更高效！

【核心功能】
• 快速开始训练：3 秒即可开始，无需复杂设置

无论你是健身新手还是达人，Interval Timer 都能满足你的需求。特别适合健身房自由重量训练、器械训练场景。

【为什么选择 Interval Timer？】
• 简单易用：界面简洁，操作直观
• 高度自定义：根据个人需求调整参数
• 完全离线：无需联网，保护隐私
• 无广告干扰：专注训练体验

立即下载，开始你的科学训练之旅！
```
