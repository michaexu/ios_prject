# UI 设计规范 - Interval Timer

## 1. 设计理念

### 1.1 设计关键词
**科技感 · 未来感 · 专业 · 简洁**

### 1.2 设计原则
1. **高对比度**：深色背景 + 亮色元素，确保健身房环境下清晰可见
2. **数据可视化**：用视觉元素展示时间流逝（进度环、波形）
3. **动态反馈**：按钮、状态变化有微动效
4. **一致性**：统一样式语言，减少认知负担

## 7. 动效规范

### 7.1 过渡时间
- **快**：0.2s（按钮反馈）
- **中**：0.3s（页面切换）
- **慢**：0.5s（加载动画）

### 7.2 缓动函数
- **标准**：`ease-in-out`
- **强调**：`cubic-bezier(0.68, -0.55, 0.265, 1.55)`

### 7.3 动效示例
- **按钮点击**：缩放 0.95 → 1.0
- **卡片出现**：透明度 0 → 1，位移 Y: 20pt → 0
- **进度环**：stroke-dashoffset 动画
