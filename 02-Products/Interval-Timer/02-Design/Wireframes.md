# 页面线框图（Wireframes）- Interval Timer

## 说明
本线框图使用 ASCII 字符绘制页面布局，展示主要元素位置和交互逻辑。

---

├─────────────────────────────────┤

### 关键元素

- **统计卡片**：全宽，显示 2-3 个关键指标

## 2. 计时器页（Timer View）

```
┌─────────────────────────────────┐
│  [← 返回]   Tabata        [ℹ️] │  ← 导航栏（方案名称）
├─────────────────────────────────┤
│                                 │
│                                 │
│         ○──────○──────○        │  ← 进度环（圆形，显示当前轮次进度）
│        ╱                      ╲ │
│       │    训练中               │ │  ← 状态文字（红色/绿色）
│       │                        │ │
│       │        00:15           │ │  ← 倒计时（72pt 大字）
│       │                        │ │
│       │  第 3 轮 / 共 8 轮     │ │  ← 轮次进度
│        ╲                      ╱ │
│         ○──────○──────○        │
│                                 │
│                                 │
│         ⏸️  暂停                │  ← 控制按钮（大圆形按钮）
│                                 │
│         ⏹️  停止                │  ← 停止按钮（小，在暂停按钮下方）
│                                 │
│                                 │
│  ●────●────●────●────●────●    │  ← 轮次指示器（小圆点，当前轮高亮）
│                                 │
└─────────────────────────────────┘
```

### 关键元素
- **进度环**：直径 280pt，线条宽度 8pt，霓虹蓝渐变
- **倒计时字体**：72pt，Bold，居中
- **控制按钮**：80pt 圆形，霓虹蓝背景
- **轮次指示器**：12pt 小圆点，当前轮为霓虹蓝，其他为灰色

### 状态变化
- **训练中**：状态文字红色，进度环霓虹蓝
- **休息中**：状态文字绿色，进度环霓虹绿
- **暂停**：状态文字黄色，进度环暂停动画
