```
│   │
│   └── 今日训练统计
│       ├── 训练次数

│       └── 连续天数
│
├── ⏱️ 计时器页（Timer）
│   ├── 顶部：返回按钮 + 方案名称
│   ├── 中间：倒计时显示（大字体）
│   │   ├── 进度环（圆形进度条）

│   ├── 底部：控制按钮
│   │   ├── 暂停/继续
│   │   └── 停止
│   └── 底部：循环进度（当前第 X/Y 轮）
│
├── 📋 方案页（Programs）
│   ├── 自定义方案
│   │   ├── "+" 新建方案按钮
│   │   └── 已保存的方案列表
│   │
│   └── 预设方案
│       └── 全部预设方案列表（可搜索）
│
├── 📊 统计页（Stats）
│   ├── 本周概览
│   │   ├── 训练次数
│   │   ├── 总训练时长
│   │   └── 连续训练天数
│   ├── 训练趋势图（柱状图/折线图）
│   └── 训练历史列表
│       └── 每条记录：日期、方案名、时长、循环数
│
└── ⚙️ 设置页（Settings）
    ├── 提醒设置
    │   ├── 声音开关
    │   ├── 震动开关
    │   └── 提示音选择
    ├── 显示设置
    │   └── 屏幕常亮开关
    ├── 关于
    │   ├── 版本号
    │   └── 隐私政策
    └── 高级
        └── 数据导出
```

## 2. 页面流程图

### 2.1 主流程（训练）
```
首页 → 点击"开始训练" / 选择预设方案 → 计时器页 → 训练完成 → 统计页
```

### 2.2 方案管理流程
```
方案页 → 点击"新建" → 设置参数 → 保存 → 方案页（更新列表）
```

### 2.3 设置流程
```
设置页 → 修改设置 → 实时生效（无需保存按钮）
```

## 3. 导航结构

### 3.1 底部标签栏（Tab Bar）
| 图标 | 标题 | 页面 |
|------|------|------|
| 🏠 | 首页 | HomeView |
| ⏱️ | 计时器 | TimerView |
| 📋 | 方案 | ProgramsView |
| 📊 | 统计 | StatsView |
| ⚙️ | 设置 | SettingsView |

- **首页 → 预设方案列表**：Push
- **方案页 → 新建/编辑方案**：Push

## 4. 数据结构

### 4.1 训练方案（Program）
```swift
struct Program {
    let id: UUID
    let name: String
    let workDuration: Int      // 训练时间（秒）
    let restDuration: Int      // 休息时间（秒）
    let rounds: Int            // 循环次数
    let isPreset: Bool         // 是否预设方案
    let createdAt: Date        // 创建时间
}
```

### 4.2 训练记录（TrainingRecord）
```swift
struct TrainingRecord {
    let id: UUID
    let programId: UUID
    let programName: String
    let date: Date
    let totalDuration: Int     // 总时长（秒）
    let completedRounds: Int   // 完成的循环数
    let totalWorkDuration: Int // 总训练时长（秒）
}
```

## 5. 用户操作流程

### 5.1 快速开始训练
1. 打开 App，进入首页
2. 显示上次训练参数（或默认 Tabata）
3. 点击"开始训练"按钮
4. 进入计时器页，自动开始倒计时

### 5.2 使用预设方案
1. 首页向下滑动，查看预设方案
2. 点击某个方案（如"Tabata"）
3. 显示方案详情（可编辑参数）
4. 点击"开始"
5. 进入计时器页

### 5.3 创建自定义方案
1. 进入"方案"页
2. 点击"+"按钮
3. 输入方案名称
4. 设置训练时间、休息时间、循环次数
5. 点击"保存"
6. 返回方案页，看到新方案

### 5.4 查看训练统计
1. 进入"统计"页
2. 查看本周数据
3. 向上滑动，查看训练趋势图
4. 继续向上，查看历史记录
